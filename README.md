## 📌 프로젝트 소개

투명지(To MyongJi)는 "To MyongJi(명지대학교 학생들을 위해)"라는 의미와 동시에 학생회비 내역을 투명하게 공개한다는 목적을 담은 학생회비 통합 관리 플랫폼입니다.

최근 발생한 명지대학교 학생회비 횡령 사건을 계기로, 기존 학생회비 관리 방식의 불투명성과 비효율성을 해결하고자 개발되었습니다.
